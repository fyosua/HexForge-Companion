Built for Performance

🪟

Transparent Overlay

Frameless, always-on-top overlay that sits on top of TFT. Cursor passes through to the game, except interactive elements.

🎮

Auto-Attach

Detects TFT process launch, resizes overlay to match the game window, and hides when TFT closes. No manual positioning needed.

🔔

System Tray Lifecycle

Minimizes to tray when TFT exits. Show/hide with system tray menu or Ctrl+Shift+H hotkey. Never in your way.

📊

Live Rank & Match Data

View your rank, LP, win/loss, match history, and aggregated stats. Pinned widget for at-a-glance data.

🏆

Leaderboard Browser

Browse Challenger, Grandmaster, and Master standings. See who's climbing and who's holding rank.

🔒

Privacy First

Zero telemetry by default. One-click GDPR wipe. All data stored locally in SQLite. Your data stays yours.

How It Works

1

Download & Run

Download the installer, run it. Works immediately in Mock mode — no API key, no configuration needed.

2

Launch TFT

The overlay detects TFT starting up, snaps to the game window, and appears as a transparent HUD on top.

3

Search Your Riot ID

Type your Riot ID (e.g. Player#TAG) to see rank, match history, stats, and live game status.

4

Pin or Browse

Pin a compact widget for at-a-glance data, or explore the full dashboard with leaderboards and server status.

Riot Games Compliance

✓ No Augment/Legend Win Rates — Blocked at the API client level. Never calculated, never displayed.
✓ No Live Scouting — Spectator endpoint never called. No opponent data during games.
✓ No Decision Dictation — Displays data only. Never tells users what to play or build.
✓ Legal Boilerplate — Riot Games legal disclaimer on every dashboard view.
✓ GDPR One-Click Wipe — Delete all stored account data with one command.
✓ Read-Only Process Detection — Standard OS APIs only. No memory scanning or DLL injection.
